# Curso Cientista de Dados – Estudos

Este repositório reúne estudos, exercícios e projetos desenvolvidos durante minha formação em Ciência de Dados.

## Conteúdos abordados
- Fundamentos de Python
- Limpeza e tratamento de dados
- Análise exploratória de dados
- Estatística aplicada
- Introdução a machine learning
- Aplicações em finanças

## Tecnologias
- Python
- Pandas
- Matplotlib
- Seaborn
- Git e GitHub

## Objetivo
Consolidar conhecimentos em Ciência de Dados por meio de projetos práticos, organização de código e versionamento profissional.
